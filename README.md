# 🚀 End-to-End Business Intelligence System

## 📌 Project Overview

This project is a complete Business Intelligence system built using MySQL and Power BI.

The system processes raw business data, transforms it into structured analytical models, and generates actionable business insights through interactive dashboards.

---

# 🎯 Business Problem

Businesses often struggle to:

- Track revenue trends
- Monitor customer behavior
- Detect churn-risk users
- Identify top-performing regions
- Understand business KPIs

This project solves these problems through an end-to-end BI pipeline.

---

# ⚙️ Features

## ✅ Data Cleaning
- Removed invalid payments
- Filtered cancelled orders
- Handled duplicate and null checks

## ✅ Data Transformation
- Revenue calculations
- User-level aggregation
- Monthly revenue analysis

## ✅ Data Warehouse
- Fact and Dimension modeling
- Star schema implementation

## ✅ KPI Layer
- Total Revenue
- Active Users
- Average Order Value

## ✅ Advanced Analytics
- Customer churn analysis
- Customer segmentation
- Revenue contribution analysis
- Top customer identification

## ✅ Dashboard
Interactive Power BI dashboard with:
- KPI cards
- Revenue trends
- Churn visualization
- Customer insights
- Country-level analysis
